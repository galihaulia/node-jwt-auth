# Node.js JWT Auth 🔐

Simple JWT-based authentication with Handlebars view rendering.

## 🛠 Stack
- Node.js
- Express.js
- JWT (jsonwebtoken)
- Handlebars (hbs)
- dotenv

## 🚀 Features
- Login/Register with JWT
- Token verification
- Session-like flow without cookies

## ⚙️ Setup
```bash
npm install
npm run dev
